# 双亲委派机制
双亲委派机制就是：当一个类加载器需要加载某个类时，它不会自己去尝试加载，而是先把加载任务委托给父类加载器去完成。如果父类加载器加载失败，才由自己来加载。

# 层级关系
1、启动类加载器（Bootstrap，最顶层，C++实现），加载 /$JAVA_HOME/lib/ 目录
2、扩展类加载器（Extension），加载 /$JAVA_HOME/lib/ext/ 或 由系统变量 java.ext.dirs 指定路径下的Jar包
3、应用程序类加载器（Application，加载classpath），加载我们平时 classpath 下的所有类
4、自定义类加载器（用户自己写的）
