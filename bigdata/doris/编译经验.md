# doris 源码编译

## mac docker 源码编译
### 1、下载master分支对应的镜像
```
docker pull apache/doris:build-env-ldb-toolchain-latest
```
### 2、启动容器
```
docker run -it \
--network=host \--name doris-docker \
-v ~/.m2:/root/.m2 \
-v /Users/wangjiang/Project/IdeaProjects/apache/doris:/root/doris \
apache/doris:build-env-ldb-toolchain-latest
```
### 3、编译
```
sh build --fe -j 8
sh build --fe -j 8
```
### 4、mac 环境端口映射 
```
docker run -it \
-p 127.0.0.1:8040:8040 \
--privileged \
--name doris-docker \
-v ~/.m2:/root/.m2 \
-v /Users/wangjiang/Project/IdeaProjects/apache/doris:/root/doris \
apache/doris:build-env-ldb-toolchain-latest
```
